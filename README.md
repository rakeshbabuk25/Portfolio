# Rakesh Babu Kancharla — Personal Portfolio

A personal portfolio website built with pure HTML and CSS. No frameworks, no build tools, no dependencies. Just a fast, clean, single-file site that loads in under a second and works on every device.

**Live site:** [rakeshbabuk25.github.io/Portfolio](https://rakeshbabuk25.github.io/Portfolio)

---

## What it covers

- **Home** — name, role, and a short intro with a call to action
- **About** — bio, core skills, and tools
- **Portfolio** — 7 projects displayed in a responsive grid, each with a custom SVG visual
- **GitHub** — live list of public repositories with descriptions
- **Contact** — email, location, GitHub, and LinkedIn

## Running locally

No installation needed.

```bash
git clone https://github.com/rakeshbabuk25/Portfolio.git
cd Portfolio
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

Or just drag `index.html` into any browser.

---

## Deploying to GitHub Pages

1. Go to **Settings → Pages** in this repository
2. Set source to **Deploy from a branch**
3. Select **main** branch, **/ (root)** folder
4. Click **Save**

The site will be live at `https://rakeshbabuk25.github.io/Portfolio` within a minute.
